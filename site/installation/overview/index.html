    Vue d'ensemble - Installation

    Cette section décrit comment installer et configurer l'environnement de la borne d'arcade.

    📦 Prérequis système

    Système d'exploitation

    • Linux (Ubuntu 20.04+ recommandé)
    • Accès root ou sudo pour certaines installations

    Logiciels requis

    Pour tous les jeux

    sudo apt update
sudo apt install -y git build-essential

    Pour les jeux Java

    sudo apt install -y openjdk-25-jdk
java -version  # Vérifier : Java 25+

    Pour les jeux Python

    sudo apt install -y python3 python3-pip
python3 --version  # Vérifier : Python 3.8+

    Installer les dépendances Python :

    pip3 install pygame

    Pour les jeux Lua

    sudo apt install -y love
love --version  # Vérifier : Love2D 11.x

    🔧 Configuration de la borne

    1. Cloner le projet

    cd ~/Bureau/Bureau/IUT/3eme\ année/Travail/SAé/SAE\ Maintenance/
git clone <url-du-repo> borne_arcade
cd borne_arcade

    2. Rendre les scripts exécutables

    chmod +x *.sh
chmod +x *.py
chmod +x projet/*/*.sh

    3. Compiler les jeux Java

    ./compilation.sh

    4. Tester un jeu

    # Java
./JavaSpace.sh

# Python
./Pong.sh

# Lua
./CursedWare.sh

    🎮 Configuration du clavier

    La borne utilise un clavier arcade avec configuration spéciale. Voir ClavierBorneArcade.java pour les mappings.

    Mapping par défaut

    Joystick J1 : - Haut/Bas/Gauche/Droite : Flèches directionnelles

    Joystick J2 : - Haut : O - Bas : L
    - Gauche : K - Droite : M

    6 touches J1 : - Ligne haute : R, T, Y - Ligne basse : F, G, H

    6 touches J2 : - Ligne haute : A, Z, E - Ligne basse : Q, S, D
